# CI/CD Pipeline Example

Here's a simple CI/CD pipeline diagram:

┌──────────────┐
│ Source Code  │
└──────────────┘
       ↓
┌──────────────┐
│  Build Job   │
└──────────────┘
       ↓
┌──────────────┐
│  Test Suite  │
└──────────────┘
       ↓
┌──────────────┐
│   Deploy     │
└──────────────┘

The pipeline ensures code quality before deployment.
