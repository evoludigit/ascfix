# Double Arrow Workflow

Demonstrating double arrow support (⇓ ⇑ ⇒ ⇐):

```
┌─────────┐
│ Start   │
└─────────┘
    ⇓
┌─────────┐
│ Process │
└─────────┘
    ⇓
┌─────────┐
│  Done   │
└─────────┘
```

Horizontal double arrows:

```
┌──────┐  ⇒  ┌──────┐  ⇒  ┌──────┐
│ Step │     │ Next │     │ End  │
└──────┘     └──────┘     └──────┘
```

Upward double arrows:

```
┌─────────┐
│ Final   │
└─────────┘
    ⇑
┌─────────┐
│ Middle  │
└─────────┘
    ⇑
┌─────────┐
│ Start   │
└─────────┘
```
